## 🎯 전체 과정 요약

---

## 📤 STEP 1: GitHub에 코드 올리기

### 1-1. GitHub 계정 만들기
1. https://github.com 접속
2. 우측 상단 "Sign up" 클릭
3. 이메일, 비밀번호 입력하여 가입

### 1-2. 새 Repository 만들기
1. 로그인 후 우측 상단 **"+"** 버튼 클릭
2. **"New repository"** 선택
3. Repository 이름: `theduckgo` 입력
4. **Public** 선택
5. **"Create repository"** 클릭

### 1-3. 코드 업로드
방법 A: 웹에서 직접 업로드 (쉬움!)
1. "uploading an existing file" 링크 클릭
2. 이 폴더의 **모든 파일**을 드래그 앤 드롭
   ```
   ✅ package.json
   ✅ index.html
   ✅ vite.config.js
   ✅ tailwind.config.js
   ✅ postcss.config.js
   ✅ .gitignore
   ✅ src/ 폴더 (전체)
   ```
3. 하단에 "Commit changes" 클릭
4. 완료!

---

## 🚀 STEP 2: Vercel로 사이트 배포

### 2-1. Vercel 계정 만들기
1. https://vercel.com 접속
2. **"Sign up with GitHub"** 클릭
3. GitHub 계정으로 로그인하여 연동

### 2-2. 프로젝트 배포
1. Vercel 대시보드에서 **"Add New..."** 버튼 클릭
2. **"Project"** 선택
3. 방금 만든 **theduckgo** repository 찾기
4. **"Import"** 클릭
5. 설정은 **아무것도 건드리지 말고** 그대로!
6. **"Deploy"** 버튼 클릭
7. 1-2분 기다리면 배포 완료! 🎉

배포되면 이런 주소를 받게 됩니다:
```
https://theduckgo-abc123.vercel.app
```

이 주소로 접속해서 사이트가 잘 작동하는지 확인하세요!

---

## 🌐 STEP 3: 가비아 도메인 연결

이제 **theduckgo.com**으로 접속하면 사이트가 보이도록 연결합니다!

### 3-1. Vercel에서 도메인 추가
1. Vercel 프로젝트 페이지에서 **"Settings"** 탭 클릭
2. 왼쪽 메뉴에서 **"Domains"** 클릭
3. **"Add"** 버튼 클릭
4. `theduckgo.com` 입력
5. **"Add"** 클릭

Vercel이 이렇게 안내해줄 겁니다:
```
DNS 설정이 필요합니다:

Type: A
Name: @
Value: 76.76.21.21

Type: CNAME  
Name: www
Value: cname.vercel-dns.com
```

### 3-2. 가비아에서 DNS 설정

1. **가비아 접속**
   - https://www.gabia.com
   - 로그인

2. **My가비아 → 서비스 관리**
   - 상단 메뉴에서 "My가비아" 클릭
   - "도메인" 섹션 찾기

3. **도메인 관리 도구**
   - `theduckgo.com` 옆의 "관리" 또는 "설정" 버튼 클릭
   - **"DNS 정보"** 또는 **"네임서버 설정"** 메뉴 찾기

4. **DNS 레코드 추가**

   **① A 레코드 추가:**
   ```
   타입(Type): A
   호스트(Name): @ (또는 공백)
   값(Value): 76.76.21.21
   TTL: 3600 (기본값)
   ```
   → "추가" 또는 "저장" 클릭

   **② CNAME 레코드 추가:**
   ```
   타입(Type): CNAME
   호스트(Name): www
   값(Value): cname.vercel-dns.com
   TTL: 3600 (기본값)
   ```
   → "추가" 또는 "저장" 클릭

5. **저장하고 완료!**

## ❓ 문제 해결

### Q1: Vercel 배포가 실패했어요
A: package.json이 제대로 업로드됐는지 확인하세요

### Q4: 비용이 드나요?
A: Vercel 호스팅은 **완전 무료**입니다!
